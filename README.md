# find-encrypt
Simple Java code encrypt file content by make it invisible

# Algorithm
    - "Let's drop the bank" 
    - "0101010000101010101..."
    - map 
        0 --> TAB
        1 --> SPACE
    - "TAB SPACE TAB TAB SPACE ..."
    - and then we have an invisible message
